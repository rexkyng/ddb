# ddBang

Search engine powered by [Next.js](https://nextjs.org/) providing [bangs](https://duckduckgo.com/bangs) from DuckDuckGo, suggestion from google, and search from Bing.

## Usage

### Search

```
/?q=<query>
```

### Suggestions

```
/suggest?q=<query>
```
